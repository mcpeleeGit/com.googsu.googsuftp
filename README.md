# GoogsuFTP - macOS FTP 관리 프로그램

GoogsuFTP는 macOS용으로 개발된 직관적이고 사용하기 쉬운 FTP 클라이언트입니다.

## 주요 기능

### 🔗 FTP 서버 연결
- FTP 서버 주소, 포트, 사용자명, 비밀번호 설정
- SSL/TLS 지원
- 연결 상태 실시간 모니터링

### 📁 파일 브라우저
- 서버의 파일과 디렉토리 탐색
- 디렉토리 이동 및 새 폴더 생성
- 파일/폴더 삭제
- 파일 권한 및 크기 정보 표시

### ⬆️⬇️ 파일 전송
- 파일 업로드 및 다운로드
- 전송 진행률 표시
- 다중 작업 동시 처리
- 전송 작업 상태 모니터링

## 시스템 요구사항

- macOS 12.0 (Monterey) 이상
- Xcode 14.0 이상 (개발용)

## 설치 및 실행

### 개발 환경에서 실행
1. 프로젝트를 클론합니다
2. Xcode에서 `googsuftp.xcodeproj`를 엽니다
3. Build & Run (⌘+R)을 실행합니다

### 앱 빌드
1. Xcode에서 Product > Archive를 선택합니다
2. 생성된 .xcarchive 파일을 사용하여 .app 파일을 만듭니다

## 사용법

### 1. FTP 서버 연결
- "연결" 탭에서 서버 정보를 입력합니다
- 서버 주소, 포트, 사용자명, 비밀번호를 설정합니다
- "연결" 버튼을 클릭합니다

### 2. 파일 탐색
- "파일" 탭에서 서버의 파일과 폴더를 탐색합니다
- 폴더를 더블클릭하여 이동합니다
- 상단의 화살표 버튼으로 상위 폴더로 이동할 수 있습니다

### 3. 파일 전송
- "전송" 탭에서 파일 업로드/다운로드를 수행합니다
- 업로드: "파일 선택" 버튼으로 로컬 파일을 선택하고 업로드합니다
- 다운로드: 파일 브라우저에서 파일을 선택하고 다운로드합니다

```
googsuftp/

│   ├── googsuftpApp.swift          # 앱 진입점
│   ├── ContentView.swift           # 메인 뷰

│   ├── ConnectionView.swift        # 서버 연결 뷰
│   ├── FileBrowserView.swift       # 파일 브라우저 뷰

│   └── Assets.xcassets/            # 앱 아이콘 및 이미지
├── googsuftp.xcodeproj/            # Xcode 프로젝트 파일
└── README.md                       # 프로젝트 설명서
```

## 기술 스택

- **SwiftUI**: 사용자 인터페이스
- **Network Framework**: FTP 연결 및 통신
- **Combine**: 반응형 프로그래밍
- **UniformTypeIdentifiers**: 파일 타입 관리

## 문제 해결

### 연결 문제
- 방화벽 설정을 확인하세요
- 서버 주소와 포트가 올바른지 확인하세요
- 사용자명과 비밀번호가 정확한지 확인하세요

### 파일 전송 문제
- 네트워크 연결 상태를 확인하세요
- 서버의 파일 권한을 확인하세요
- 충분한 디스크 공간이 있는지 확인하세요
